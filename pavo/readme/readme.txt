
FILE STRUCTURE
- index.html holds the entire content with Tailwind CSS utility classes
- article.html, privacy.html and terms.html are additional pages
- Tailwind CSS is imported via CDN link
- css/styles.css custom css styling
- js/scripts.js custom js code
- images folder contains all the images
- The rest are files specific to different frameworks and dependencies


FRAMEWORKS & DEPENDENCIES
- Tailwind CSS https://tailwindcss.com/
- jQuery https://jquery.com/ 
- jQuery Easing https://jqueryui.com/easing/
- Magnific Popup https://dimsemenov.com/plugins/magnific-popup/
- Swiper https://swiperjs.com/
- Font Awesome for icons https://fontawesome.com
